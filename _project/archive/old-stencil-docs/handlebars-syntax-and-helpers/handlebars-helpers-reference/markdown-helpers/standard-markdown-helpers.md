The following standard helper is available to convert markdown.

## {{markdown}}

Block helper that converts a string of inline markdown to HTML.

#### Parameters

* `context` {Object}
* `options` {Object}
* `returns` {String}

#### Example

```html
{{#markdown}}
# Foo
{{/markdown}}
//=> <h1>Foo</h1>
```
